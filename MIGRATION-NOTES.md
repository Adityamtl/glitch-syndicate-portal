# Glitch Syndicate Portal - React Version

This directory contains both the original static HTML version and the new React version.

## Files Overview

### Original Static Site

- `index.html` - Original static homepage
- `about.html` - Original about page
- `css/styles.css` - Original styles
- `js/app.js` - Original JavaScript

### React Application

- `index-react.html` - React app HTML template
- `src/` - React source code
- `package.json` - Dependencies
- Configuration files (vite, tailwind, postcss)

## Running the React Version

```bash
npm install
npm run dev
```

See `README-REACT.md` for full documentation.

## Key Improvements in React Version

✅ Component-based architecture
✅ Smooth page transitions with Framer Motion
✅ Better state management
✅ Improved animations and effects
✅ Tailwind CSS for easier styling
✅ Mobile-responsive design
✅ Better performance
✅ Easy to maintain and extend
